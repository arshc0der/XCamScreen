# üìò **XCamScreen ‚Äî User Guide**

XCamScreen is a desktop application that allows you to record:

* Camera (Webcam)
* Screen
* Camera + Screen (Picture-in-Picture)
* And capture photos

This guide explains how to use each feature step-by-step.

---

## üèÅ **Starting the Application**

When XCamScreen opens, you will see the main interface with a live preview in the middle and recording controls below.

---

## üîÑ **Choosing Recording Mode**

At the upper right corner, you will see a **Mode** drop-down list:

| Mode                   | Description                                                                          |
| ---------------------- | ------------------------------------------------------------------------------------ |
| **Camera**             | Records only the webcam                                                              |
| **Screen**             | Records only your computer screen                                                    |
| **Cam + Screen (PiP)** | Records screen and webcam simultaneously ‚Äî webcam appears in a small floating window |

### How to Choose:

1. Click the **Mode** drop-down.
2. Select the mode you want.

---

## üé• **Camera and Input Settings**

At the bottom left:

| Setting        | Purpose                            |
| -------------- | ---------------------------------- |
| **Camera**     | Select your webcam device          |
| **Resolution** | Choose video size (e.g., 1280√ó720) |

If you don‚Äôt see video, select another camera device.

---

## üé§ **Microphone and Audio**

Next to camera settings:

| Setting            | Purpose                             |
| ------------------ | ----------------------------------- |
| **Microphone**     | Select your audio input device      |
| **Bitrate**        | Adjust video quality                |
| ‚úî **Record Audio** | Enable/disable microphone recording |

If you only want video recording with no sound, uncheck **Record Audio**.

---

## üì∫ **Screen Recording Settings**

When Screen or PiP mode is selected:

| Setting    | Purpose                                                   |
| ---------- | --------------------------------------------------------- |
| **Screen** | Select which monitor to record (Screen 1 / Screen 2 etc.) |
| **FPS**    | Choose frames per second (higher = smoother video)        |

---

## ‚öôÔ∏è **Extra Options**

| Option                                    | Description                                        |
| ----------------------------------------- | -------------------------------------------------- |
| ‚úî **Show Live Preview**                   | Display webcam preview in the app                  |
| ‚úî **Hardware Acceleration**               | Improves recording performance                     |
| ‚óª **Show webcam overlay while recording** | Displays PiP webcam on top of screen recording     |

> Tip: For tutorial videos, use **Bottom-Right + Round** for a clean overlay.

---

## ‚ñ∂Ô∏è **Starting and Stopping Recording**

| Button                             | Function                        |
| ---------------------------------- | ------------------------------- |
| **Start Recording (Green button)** | Begins video recording          |
| **Stop Recording (Red button)**    | Stops recording and saves video |

### Where are videos saved?

After stopping recording, the app automatically saves the file in the folder you set inside the recording settings (or default folder if none is set).

---

## üì∏ **Capturing a Photo**

To take a snapshot of your webcam:

* Click **Capture Photo** at any time during preview or recording.

The captured image will be saved automatically to the output folder.

---

## üß∞ **Best Settings for Smooth Recording**

| Use Case                     | Recommended Settings                  |
| ---------------------------- | ------------------------------------- |
| Full screen tutorial videos  | Screen mode ‚Üí 60 FPS ‚Üí 8 Mbps         |
| Face-cam recording           | Camera mode ‚Üí 720p / 1080p            |
| PiP teaching / gaming videos | Cam + Screen ‚Üí 30 FPS ‚Üí Round overlay |

---

## üìù **Troubleshooting**

| Issue                                          | Solution                                                           |
| ---------------------------------------------- | ------------------------------------------------------------------ |
| Camera preview is black                        | Select correct **Camera** device                                   |
| No audio in video                              | Enable **Record Audio** and choose the right **Microphone**        |
| Laggy recording                                | Enable **Hardware Acceleration**, reduce **Resolution** or **FPS** |
| Webcam PiP not visible during screen recording | Enable **Show webcam overlay while recording**                     |
