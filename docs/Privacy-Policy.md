## **How We Use and Secure Your Data**

All processing occurs **locally on your device** and is handled only for core functionality, such as:

* Webcam recording
* Screen recording
* Picture-in-Picture (PiP) overlay
* Audio recording when enabled by the user

No data is transmitted externally during any operation.
XCamScreen does not include any code for analytics, advertisements, telemetry, or monitoring.

## **Permissions and Access**

XCamScreen requests access only for features necessary to function:

| Permission            | Purpose                                               |
| --------------------- | ----------------------------------------------------- |
| Camera                | Capture video in Camera & PiP mode                    |
| Microphone (optional) | Record audio when user enables it                     |
| Screen / Display      | To record desktop content                             |
| File System           | To save recordings/photos to a user-selected location |

The application does **not** access:

* Internet or network activity
* GPS / location
* Contacts / messages
* Personal user documents not selected for export

## **Third-Party Services**

XCamScreen does **not integrate with** and does **not send data to**:

* Advertising platforms
* Cloud storage providers
* Analytics or tracking services
* Third-party data processors

The software is completely standalone and offline.

## **Your Rights**

Because all recordings and data are stored locally, **you remain the sole owner of your content.** You may:

* View recordings anytime
* Move or delete video/photo files manually
* Uninstall the application at any time without involving the developer

No request is required from the developer to access, export, or delete your data.
